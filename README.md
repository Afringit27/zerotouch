# Zero-Touch AI-Based Multi-Sensor Safety Trigger System

This project detects danger automatically using multiple sensors and AI-based decision logic.

## Components Used
ESP32 microcontroller
MPU6050 motion sensor
Sound sensor
Vibration sensor
MAX30102 heart rate sensor

## Technologies
IoT
Artificial Intelligence
Embedded Systems
Python UI

## Features
Real-time monitoring
Automatic danger detection
Zero-touch alert generation
User interface dashboard

## How to Run

1. Upload Arduino code to ESP32
2. Connect sensors properly
3. Run Python UI
4. Observe SAFE or DANGER output
